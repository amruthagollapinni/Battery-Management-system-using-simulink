# Battery Management System (BMS) Using MATLAB/Simulink

## Overview
This project presents a Battery Management System (BMS) simulation developed using MATLAB/Simulink. The model monitors key battery parameters such as State of Charge (SOC), State of Health (SOH), Open Circuit Voltage (OCV), Terminal Voltage, Current, and Temperature to evaluate battery performance during operation.

The project demonstrates the basic functions of a Battery Management System and provides simulation results for battery monitoring and analysis.

---

## Objectives

- Simulate a Battery Management System (BMS) using MATLAB/Simulink.
- Estimate the State of Charge (SOC) of the battery.
- Estimate the State of Health (SOH) of the battery.
- Monitor battery voltage and Open Circuit Voltage (OCV).
- Monitor battery current.
- Implement a simple thermal model for battery temperature estimation.
